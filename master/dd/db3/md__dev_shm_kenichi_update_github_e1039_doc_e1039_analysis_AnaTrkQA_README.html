AnaTrkQA

Work on progress anslysis moudle for quality assurance of tracks.

Building

The analysis code is located in e1039-analysis/AnaTrkQA/src. The following commands set up the shell environment and compile the code;

cd e1039-analysis/AnaTrkQA
source setup.sh
cd build
cmake-ana-TrkQA
make install

Running the analysis macro

Run Fun4Sim_TrkQA.C macro in macors\area
